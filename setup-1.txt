system-2:
=========
1. Amazon-linux-extras install java-openjdk11.
 
2. wget https://dlcdn.apache.org/tomcat/tomcat-9/v9.0.65/bin/apache-tomcat-9.0.65.tar.gz

3. tar -xvzf apache-tomcat-9.0.65.tar.gz

4. If you want to change the port of tomcat:
    cd apache tomcat->cd conf->vi server.xml
    Change the connector port to 80 along with executor port to 80 which below to the connector port and remove the comment for the connector port.

5. vi /tomcat/webapps/manager/context.xml
   comment-out value tag   <!--     /-->   (Comment out the tag by giving the hyphens after the slash.)

6. vi /tomcat/conf/
       <role rolename="admin-gui"/><role rolename="manager-gui"/><user username="admin" password="admin" roles="admin-gui,manager-gui"/>

7. Restart tomcat.

8. Copy the public ip address and paste it in the browser.
Note: Now the tomcat runs on 80 port itself and no need to mention the port alngside the ip address.

9. If you have given the port as 8081 or 8080 then you have to mention the port with the ip address.

10. Check whether tomcat manager aand host-manager are working by entering the username and password.

11. Now install an application like jenkins in the webapps and check if it's working or not.
